Privacy Policy

Pengumpulan Data

Di YTCloud Downloader, privasi Anda adalah prioritas kami. Kami tidak menyimpan catatan video atau audio apa pun yang Anda unduh, dan kami tidak menyimpan riwayat pencarian pengguna di server kami.

Penggunaan Layanan Pihak Ketiga

Layanan kami bertindak sebagai perantara yang mengirimkan URL yang Anda masukkan ke API pihak ketiga untuk mengekstrak tautan unduhan. Kami tidak menghosting file media yang dilindungi hak cipta di server kami sendiri.

Cookies

Saat ini kami tidak menggunakan cookies pelacakan untuk memantau aktivitas pribadi Anda. Fungsionalitas web sepenuhnya berjalan di browser Anda.

Persetujuan

Dengan menggunakan situs web kami, Anda dengan ini menyetujui Kebijakan Privasi kami dan menyetujui syarat-syaratnya.
